clearAccessToken method

Future<void> clearAccessToken (
  1. {@required AccessToken accessToken}
)

Clears the provided accessToken.

Note: When failing to access a resource with an obtained token, call the clearAccessToken method to clear the invalid token before calling WLAuthorizationManager.obtainAccessToken to obtain a new access token. This is a non blocking call and can be awaited

Implementation

Future<void> clearAccessToken({@required AccessToken accessToken}) async {
  var accessTokenMap =
      AccessToken.getAccessTokenMap(accessToken: accessToken);
  _channel.invokeMethod(WLAUTHORIZATIONMANAGER_CLEARACCESSTOKEN,
      <String, dynamic>{ACCESSTOKEN: accessTokenMap});
}
